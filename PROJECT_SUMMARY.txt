                      DEPLOYMENT STEPS

🚀 DEPLOY NOW (5 minutes):

  1. Go to: https://railway.app/new
  
  2. Click: "Deploy from GitHub repo"
  
  3. Select: ksarrafi/railway-vault
  
  4. Add volume:
     • Name: data
     • Mount: /vault/data
  
  5. Generate token:
     $ openssl rand -base64 32
  
  6. Set in Railway:
     Variables → VAULT_TOKEN → [paste token]
  
  7. Save token locally:
     $ echo "[token]" > ~/railway-vault-token.txt
     $ chmod 600 ~/railway-vault-token.txt
  
  8. Test:
     $ curl https://your-vault.railway.app/health
     Expected: {"ok": true, "backend": "railway", "version": "1.0.0"}

  ✅ Monitor for 24-48 hours
  ✅ Remove old environment variables

Example (Python):
  from vault_client import VaultClient
  vault = VaultClient()
  DATABASE_URL = vault.get("myapp:database_url")

Example (Node.js):
  const VaultClient = require('./vault_client');
  const vault = new VaultClient();
  const DATABASE_URL = await vault.get('myapp:database_url');

Base URL: http://kmac-vault.railway.internal:9999
Auth: Bearer [token]

📍 Health Check
  GET /health
  Response: {"ok": true, "backend": "railway", "version": "1.0.0"}

📍 Set Secret
  POST /set
  Body: {"key": "myapp:db_url", "value": "postgresql://..."}
  Response: {"ok": true, "key": "myapp:db_url"}

📍 Get Secret
  GET /get/myapp:db_url
  Response: {"key": "myapp:db_url", "value": "postgresql://..."}

📍 List Keys
  GET /list
  Response: {"keys": ["myapp:db_url", "myapp:api_key", ...]}

📍 Delete Secret
  POST /delete/myapp:db_url
  Response: {"ok": true, "key": "myapp:db_url"}
